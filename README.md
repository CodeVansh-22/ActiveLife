# ActiveLife

ActiveLife is a web-based fitness management system that helps users track BMI, receive diet suggestions, select workouts, and book trainer appointments. It centralizes health records and scheduling into one platform, improving gym management efficiency and providing users with structured, convenient fitness guidance online.

# Clone the repository

```bash
git clone https://github.com/your-username/activelifes.git
cd activelifes
```

# Install dependencies

```bash
pip install -r requirements.txt
```

# Run the application

```bash
python app.py
```

# Access the application

Open your web browser and navigate to http://localhost:5000 to access the application.

# Tech Stack

- Frontend: Angular
- Backend: Python,Flask
- Database: MySQL
- AI: Gemini

#Frontend Setup

1. Install Angular CLI

```bash
npm install -g @angular/cli
```

2. Create a new Angular project

```bash
ng new activelife-frontend
```

3. Navigate to the project directory

```bash
cd activelife-frontend
```

4. Install dependencies

```bash
npm install bootstrap
```

5. Run the application

```bash
ng serve
```

6. Access the application
   Open your web browser and navigate to http://localhost:4200 to access the application.
